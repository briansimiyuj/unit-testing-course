A unit test is a small piece of code that tests a specific function or method in your project. It is used to ensure that the code is working as expected and to catch any errors or bugs that may occur.

Unit tests are done in isolation, meaning that they test specific parts of the code without depending on other parts of the code. 

Unit tests only care the functionality of the code, not the logic behind it; input and output are the only things that matter.
